# Cook Order ~ FOR YOU ~

## 目的

企画設計や UI デザインから、システムの構築（CI/CD、デプロイ環境を含む）までを一通り作成してみる。

## 概要

料理注文アプリ

- 作った人の顔写真を掲載
- チップを提供できる機能を追加する予定

## 機能

### クライアントアプリ

- ログイン機能
  - 注文内容確認
  - 過去の注文リスト
  - ゲストユーザ対応
- 料理注文機能
- チップ送付機能

将来的にマネジメントアプリを作成し、料理の追加編集を可能にする。

## デプロイ先

将来的に AWS または GCP でアプリを運用していく予定。

- [Vercel app](https://cook-order.vercel.app/)
